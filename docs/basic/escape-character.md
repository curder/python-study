# 转义字符

## 无法直接表示的字符

| 转义字符 | 表达的字符 | 备注                   |
|------|-------|----------------------|
| `\n` | 换行    | newline 光标移动到下一行的开头处 |
| `\t` | 水平制表符 | tab 光标移动到下一组4个空格的开头处 |
| `\r` | 回车    | return 光标移动到本行的开头    |
| `\b` | 退格    | backspace 回退一个字符     |

## 字符串中特殊用途字符

| 转义字符 | 表达的字符 |
|------|-------|
| `\\` | 反斜杠   |
| `\'` | 单引号   |
| `\"` | 双引号   |
