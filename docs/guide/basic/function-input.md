# input 输入函数

- input 函数接受用户输入的值。

    ```python
    present = input('What\'s your name: ')

    print('Your name is ', present)  # Your name is Curder
    ```

- 获取用户输入的整型，并求和

    ```python
    a = input('Give me a number: ')
    b = input('Give me another number: ')

    print(int(a) + int(b))
    ```
