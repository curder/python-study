# 元祖操作

## 计算元祖长度

使用 `len()` 函数可以计算元祖长度。

```python
print(len((1, 2, 3, 4)))  # 4
```

## 获取最大元素

使用 `max()` 函数获取元祖中的最大元素。

```python
print(max((1, 2, 3, 4)))  # 4
```

## 获取最小元素

使用 `min()` 函数获取元祖中的最小元素

```python
print(min((1, 2, 3, 4)))  # 1
```

## 重复元祖元素

可以使用 `*` 操作元祖，使其重复指定个数的元祖元素。

```python
t = (10,)
print(t * 10)  # (10, 10, 10, 10, 10, 10, 10, 10, 10, 10)
```
